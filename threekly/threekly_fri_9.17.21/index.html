1. A podcast I’ve been binging

Anthony Pomliano

This video does a great job explaining it, but in short, it deals with Set Theory. Imagine a set that contains all sets that do not contain itself. Does this set contain itself? The rules of the set says it both should, and should not…

2. A project I’ve come to love

Loot Project

Loot Project is the brain child of Dom Hoffman, the creator of the once loved Vine. An entirely on-chain NFT (using SVG – this alone is noteworthy), this project serves as the starting place for myriad games and communities. The project defines a set of “loot” that each holder has. Being on-chain and in a decentralized environment, Loot Project allows anyone to ideate and grow the game or community. Some interesting spin offs are games, other loot components, and even synthetic loot.

3. A book I’ve been reading

Algorithms for Everyday Life by Brian Christian and Tom Griffiths

This book explores algorithms and how they can apply to our everyday understanding of decision making. While having knowledge of some algorithms helps, the authors do a great job explaining to people without technical knowledge. From when to stop (stopping problem and secretaries problem) to recipes, this book breaks down how humans can use “machine” decision making to streamline choices and ultimately pick the best every time.
